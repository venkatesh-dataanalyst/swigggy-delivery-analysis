# Swiggy Delivery Data Analysis

This project analyzes Swiggy restaurant data using SQL and Python.

## Tools Used
- Python
- Pandas
- SQLite
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Top cities with most restaurants
- Price distribution
- Delivery time distribution
- Top rated restaurants

## Insights
- Most restaurants fall in ₹200–₹500 price range
- Delivery times mostly between 40–60 minutes
- Some restaurants have ratings close to 5.0
